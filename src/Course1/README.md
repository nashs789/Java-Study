## 📌 최상위 클래스 Object

클래스가 상속 받는 부모 클래스가 없다면 묵시적으로 extends Object 라고 보면 된디.
(컴파일러에 의해서 동작되지 않을까?)

일관성을 갖는 프로그래밍을 위해서 가장 기본적인 기능만 하도록 만들어진 최상위 객체이다.  
모든 객체를 참조할 수 있는 다형성 지원한다. (객체지향 기본적인 메커니즘 제공)
